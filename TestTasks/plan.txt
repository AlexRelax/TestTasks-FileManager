Приложение реализовано на WinForms. 
В окне нужно выбрать файл для занесения результатов и указать директорию для сканирования. Выбирая файл/директорию в левой панели можно просмотреть данные об этом файле/директории в правой панели 


Выбор каталога для сканирования 

Модель данных (PropertyInfo)
Обход дерева файловой системы и занесение результатов в List< PropertyInfo>. (public class FileSystemEnumerate)

Заполнение TreeView данными из List< PropertyInfo>.
Убрал мигание при добавлении узла к дереву (BufferedTreeView : TreeView)
Вывод основных параметров (имя, дата создания, …) по клику

Реализовал два потока 
1.	"поток сбора информации" - Обход дерева файловой (private void EnumerateDirs()
2.	"поток занесения результатов в дерево" – (private void ShowDirs(object o))

Синхронизация потоков (MonitorWrapper.cs)

Третий поток "поток занесения результатов в XML файл" (public class FileSystemToXml)

Интерфейс
Выбор каталога записи результатов в XML файл

Добавление и вывод атрибутов

Добавление и вывод владельца

Добавление и вывод прав текущего пользователя
